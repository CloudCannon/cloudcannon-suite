Welcome to the prerequisites guide. Use this guide to get your development machine setup.

Node & NPM

To check if you have Node.js installed, run this command in your terminal:

$ node -v

To confirm that you have npm installed you can run this command in your terminal:

$ npm -v

If either of these commands fail, follow the npm install guide.

Install the gulp-cli

Installing the gulp CLI allows you to run the gulp command in your terminal:

$ npm install --global gulp-cli

Now that you have development machine setup you can move to installing the CloudCannon Suite.
